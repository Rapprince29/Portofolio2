## ✦ System Overview

This repository houses the source code for my **High-End Interactive Portfolio**. Designed not just as a website, but as a "Digital Operating System" experience—blending cinematic motion engineering, strict minimalist typography (Neo-Brutalism), and extreme performance.

## 💎 God-Tier Features

- **Magnetic Tech Arsenal:** Interactive utility tiles that simulate physical gravity and pull toward the user's cursor.
- **Cinematic Focus Mode:** Project tiles auto-dim surrounding non-focus elements to draw deep attention to selected case studies.
- **Identity Decryption Preloader:** Kinetic typography engine that "decrypts" and scrambles characters on initial boot.
- **Glass-Mist Architecture:** Heavy use of layered translucent panels (`backdrop-blur`) mixed with a sub-layer noise grain for premium tactical depth.
- **Hidden Command Terminal:**
  - Standard users see a website.
  - Engineers can press <kbd>Ctrl</kbd> + <kbd>K</kbd> to open a hidden Unix-like system terminal. 
  - Try commands like `whoami`, `projects`, `skill`, and `sudo hire yoga`.

## 📂 Project Structure

This is a monolithic React Application housed inside the `.p-next/` directory.

```bash
/p-next
  ├── src/
  │   ├── app/           # Next.js App Router (page.tsx, layout.tsx)
  │   └── components/    # Reusable UI Modules (Navbar, NeuralBackground)
  ├── public/            # Static Assets
  └── tailwind.config.ts # Core styling tokens
```

## 🚀 Boot Sequence (Local Run)

To test the ecosystem locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/Rapprince29/Portofolio2.git
   ```
2. Enter the working directory:
   ```bash
   cd Portofolio2/p-next
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Ignite the server:
   ```bash
   npm run dev
   ```
5. Navigate to `http://localhost:3000`
